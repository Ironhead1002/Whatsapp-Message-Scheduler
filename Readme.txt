This is a whatsapp message scheduler which can send written message on desired time.

steps to use:-

1) Install python in your system and give path.
2) Then run the main.py file.
3) Enter the asked data and that is it.

Note:

It is only made for Indian mobile numbers (Starting from +91).

Your whatsapp web needs to be registered with the browser.
